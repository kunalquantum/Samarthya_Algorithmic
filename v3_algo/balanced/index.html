About Balanced Binary Tree

Approach

  • A binary tree is balanced if the height difference between left and right subtrees of every node is not more than 1.
  • We use a recursive approach:
    • Calculate the height of left and right subtrees for each node
    • Check if the absolute difference between heights is at most 1
    • Recursively check if both left and right subtrees are also balanced
  • The base case is when we reach a null node (height = 0, considered balanced).

Time and Space Complexity

  • Time Complexity: O(n), where n is the number of nodes in the tree
  • Space Complexity: O(h), where h is the height of the tree (due to recursion stack)
