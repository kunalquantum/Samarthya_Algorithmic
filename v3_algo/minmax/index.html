About Min-Max Finding

Single Scan Approach

Finds both minimum and maximum elements in an unsorted array using a single traversal.

Algorithm Details:

  • Time Complexity: O(n)
    • Single pass through the array
    • Each element is compared at most twice
  • Space Complexity: O(1)
    • Only needs two variables for min and max
    • No additional space required

Key Steps:

  • Initialize min and max with first element
  • For each element in array:
    • If element < min: Update min
    • If element > max: Update max
  • Final min and max values give the result

Example:

Input Array: [64, 34, 25, 12, 22, 11, 90]

Minimum Element: 11 (at position 5)

Maximum Element: 90 (at position 6)

Explanation: Single scan identifies both extremes.
