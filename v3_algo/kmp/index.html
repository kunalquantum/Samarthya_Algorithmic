About KMP Pattern Searching

Knuth-Morris-Pratt Algorithm

An efficient string matching algorithm that uses pattern preprocessing to achieve linear time complexity.

Algorithm Details:

  • Time Complexity: O(n + m)
    • n: length of text
    • m: length of pattern
    • LPS array construction: O(m)
    • Pattern searching: O(n)
  • Space Complexity: O(m)
    • Only needs LPS array of pattern length
    • No additional space growth with text size

Key Steps:

  • Preprocessing:
    • Build LPS (Longest Proper Prefix which is also Suffix) array
    • Use pattern to identify potential skip positions
  • Pattern Matching:
    • Compare text and pattern characters
    • On mismatch, use LPS array to skip redundant comparisons
    • Continue until pattern found or text end reached

Example:

Text: "ABABDABACDABABCABAB"

Pattern: "ABABCABAB"

LPS Array: [0,0,1,2,0,1,2,3,4]

Pattern found at index: 10

Explanation: KMP algorithm efficiently finds the pattern by avoiding redundant comparisons using the LPS array.
