Algorithm Steps

Two Sum Problem

Finds two numbers in the array that add up to a target sum.

  • Time Complexity: O(n)
  • Space Complexity: O(n)

Kadane's Algorithm

Finds the contiguous subarray with the largest sum.

  • Time Complexity: O(n)
  • Space Complexity: O(1)

Dutch National Flag

Sorts an array containing only 0s, 1s, and 2s in a single pass.

  • Time Complexity: O(n)
  • Space Complexity: O(1)

Merge Sorted Arrays

Merges two sorted arrays into a single sorted array.

  • Time Complexity: O(n + m)
  • Space Complexity: O(n + m)

Leaders in Array

Finds all elements that are greater than all elements to their right.

  • Time Complexity: O(n)
  • Space Complexity: O(1)
