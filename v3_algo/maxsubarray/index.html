About Kadane's Algorithm

Maximum Subarray Problem

Finds the contiguous subarray within a one-dimensional array of numbers which has the largest sum.

Algorithm Details:

  • Time Complexity: O(n)
    • Single pass through the array
    • Maintains current sum and maximum sum seen so far
  • Space Complexity: O(1)
    • Only needs to store current sum and maximum sum
    • Additional space used only for visualization

Key Concepts:

  • At each position, decide whether to:
    • Start a new subarray (current element)
    • Extend the existing subarray (add current element)
  • If current sum becomes negative, reset it to 0
  • Keep track of maximum sum seen so far

Example:

Input Array: [-2, 1, -3, 4, -1, 2, 1, -5, 4]

Maximum Subarray: [4, -1, 2, 1]

Maximum Sum: 6

Explanation: The subarray [4, -1, 2, 1] has the largest sum = 6.
