About Tree Traversal Methods

Inorder Traversal

Visit order: Left → Root → Right

  • First, visit all nodes in the left subtree
  • Then, visit the root node
  • Finally, visit all nodes in the right subtree

For a binary search tree, inorder traversal retrieves nodes in ascending order.

Preorder Traversal

Visit order: Root → Left → Right

  • First, visit the root node
  • Then, visit all nodes in the left subtree
  • Finally, visit all nodes in the right subtree

Useful for creating a copy of the tree or prefix expression evaluation.

Postorder Traversal

Visit order: Left → Right → Root

  • First, visit all nodes in the left subtree
  • Then, visit all nodes in the right subtree
  • Finally, visit the root node

Useful for deleting a tree or postfix expression evaluation.

Time and Space Complexity

  • Time Complexity: O(n), where n is the number of nodes in the tree
  • Space Complexity: O(h), where h is the height of the tree (due to recursion stack)

Example Tree

        1
       / \

Preorder traversal: 1, 2, 4, 5, 3, 6
