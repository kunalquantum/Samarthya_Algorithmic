About Diameter of Binary Tree

Approach

  • The diameter of a binary tree is the length of the longest path between any two nodes in the tree.
  • This path may or may not pass through the root.
  • We use a recursive approach:
    • For each node, calculate the height of its left and right subtrees
    • The path through the current node = left height + right height
    • Keep track of the maximum path length seen so far
  • The base case is when we reach a null node (height = 0).

Time and Space Complexity

  • Time Complexity: O(n), where n is the number of nodes in the tree
  • Space Complexity: O(h), where h is the height of the tree (due to recursion stack)

Example

Tree:           1
               / \
              2   3
             / \
            4   5

Diameter: 3 (path from node 4 to node 3 through nodes 2 and 1)
