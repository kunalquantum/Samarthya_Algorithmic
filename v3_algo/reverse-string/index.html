About String Reversal using Stack

Algorithm

  1. Initialize an empty stack.
  2. Push each character of the input string onto the stack, from left to right.
  3. Once all characters are pushed, pop each character from the stack and append it to the result string.
  4. Since stack follows Last-In-First-Out (LIFO) principle, the characters will be popped in reverse order.
  5. The resulting string will be the reverse of the original string.

Time and Space Complexity

  • Time Complexity: O(n), where n is the length of the string
  • Space Complexity: O(n) for storing all characters in the stack

Why Use a Stack?

A stack is perfect for reversing a string because of its Last-In-First-Out (LIFO) property. The last character pushed onto the stack will be the first one popped off, naturally creating the reverse order of the original string.

While there are other ways to reverse a string (like using a reverse loop or built-in functions), using a stack demonstrates an important data structure concept and is a classic example of how stacks can be used to solve problems.

Examples

Example 1

Input: "Hello"

Output: "olleH"

1. Push: H, e, l, l, o

2. Pop: o, l, l, e, H

Example 2

Input: "Algorithm"

Output: "mhtiroglA"

1. Push: A, l, g, o, r, i, t, h, m

2. Pop: m, h, t, i, r, o, g, l, A
