About Two Pointer Technique

Find Pairs with Target Sum

Uses two pointers to find pairs of numbers in a sorted array that sum up to a target value.

Algorithm Details:

  • Time Complexity: O(n)
    • Single pass through the array
    • Each pointer moves at most n steps
  • Space Complexity: O(1)
    • Only needs two pointers
    • Additional space used only for storing results

Key Concepts:

  • Two Pointer Movement:
    • Left pointer starts from beginning
    • Right pointer starts from end
    • Move based on sum comparison with target
  • Pointer Movement Rules:
    • If sum > target: Move right pointer left
    • If sum < target: Move left pointer right
    • If sum = target: Record pair and move both pointers

Example:

Input Array: [1, 2, 3, 4, 5, 6, 7, 8, 9]

Target Sum: 10

Found Pairs: [1,9], [2,8], [3,7], [4,6]

Explanation: Each pair sums up to the target value of 10.
