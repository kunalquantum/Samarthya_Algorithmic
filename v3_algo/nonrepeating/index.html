About the Algorithm

First Non-Repeating Number

This algorithm finds the first number in the array that appears exactly once (non-repeating).

Time and Space Complexity:

  • Time Complexity: O(n) - We need to traverse the array twice
    • First pass: Count frequency of each element
    • Second pass: Find first element with frequency 1
  • Space Complexity: O(n) - We need a hash map to store frequency counts

Example:

Input Array: [7, 2, 3, 2, 4, 3, 7, 5]

First Non-Repeating Number: 4

Explanation: 4 and 5 appear only once, but 4 appears first in the array.
