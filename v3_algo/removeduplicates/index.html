About Remove Adjacent Duplicates

Stack-based Solution

Removes all adjacent duplicate characters in a string until no such duplicates exist.

How it Works:

  • Uses a stack to keep track of characters
  • Compares current character with top of stack
  • If they match, removes both (pop from stack)
  • If different, adds current character to stack
  • Final stack contents form the result string

Algorithm Details:

  • Time Complexity: O(n)
    • n: length of input string
    • Each character is processed once
    • Stack operations are O(1)
  • Space Complexity: O(n)
    • Stack can store up to n characters
    • Result string storage

Example:

Input: "abbaca"

Process:

  1. Push 'a' → Stack: [a]
  2. Push 'b' → Stack: [a,b]
  3. Push 'b' → Stack: [a,b,b] → Pop 'b','b' → Stack: [a]
  4. Push 'a' → Stack: [a,a] → Pop 'a','a' → Stack: []
  5. Push 'c' → Stack: [c]
  6. Push 'a' → Stack: [c,a]

Result: "ca"
