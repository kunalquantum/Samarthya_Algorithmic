About Symmetric Tree

Approach

  • A binary tree is symmetric if its left and right subtrees are mirror images of each other.
  • Two trees are mirrors if:
    • Their root values are equal
    • The left subtree of each tree mirrors the right subtree of the other tree
  • We use recursive comparison of corresponding nodes in the left and right subtrees.

Time and Space Complexity

  • Time Complexity: O(n), where n is the number of nodes in the tree
  • Space Complexity: O(h), where h is the height of the tree (due to recursion stack)

Example

Symmetric Tree:       1
                    / \
                   2   2
                  / \ / \
                 3  4 4  3

Not Symmetric:       1
                    / \
                   2   2
                    \   \
                     3   3
