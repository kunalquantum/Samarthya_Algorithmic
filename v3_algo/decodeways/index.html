About Decode Ways

Dynamic Programming Approach

  • Use a DP array where dp[i] represents the number of ways to decode the substring ending at index i
  • For each position i, we can decode in two ways:
    • Single digit: If digit at i is not '0', add dp[i-1] to dp[i]
    • Two digits: If digits at (i-1,i) form a valid number (10-26), add dp[i-2] to dp[i]
  • Base cases:
    • dp[0] = 1 if first digit is not '0'
    • dp[1] depends on both single and two-digit possibilities

Time and Space Complexity

  • Time Complexity: O(n) - where n is the length of the input string
  • Space Complexity: O(n) - to store the DP array

Example

Input: "12"

Can be decoded as:

  • "1 2" → "A B" (1=A, 2=B)
  • "12" → "L" (12=L)

Therefore, there are 2 ways to decode "12"
